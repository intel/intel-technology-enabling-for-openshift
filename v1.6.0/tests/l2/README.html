Verifying Intel Hardware Feature Provisioning

Introduction

After provisioning Intel hardware features on RHOCP, the respective hardware resources are exposed to the RHOCP cluster. The workload containers can request these resources. The following sample workloads help verify if these resources can be used as expected. These sample workloads container images are built and packaged on-premises through RHOCP BuildConfig and pushed to the embedded repository through RHOCP ImageStream.
